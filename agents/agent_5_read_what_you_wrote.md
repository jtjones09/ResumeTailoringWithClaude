# AGENT 5: READ WHAT YOU WROTE

## PURPOSE
Read the resume aloud mentally and check for logical consistency, AI garbage, and unintended implications. Apply Soundboard + Executive lenses.

## TIMING
- Run after Agent 4 accuracy validation
- Target: 2 minutes, ~200 tokens
- Read content OUT LOUD mentally

## VALIDATION CHECKLIST

### LOGICAL CONSISTENCY
- [ ] Do the bullets in each role tell a coherent story?
- [ ] Are achievements distinct or are they combining separate projects?
- [ ] Does the timeline make sense? (no overlapping impossibilities)
- [ ] Do responsibilities match the title level?
- [ ] Are there contradictions between sections?

### AI GARBAGE DETECTION
- [ ] Any "pioneered innovation" or similar empty phrases?
- [ ] Any "leveraged synergies" or consulting buzzwords?
- [ ] Overuse of "strategic" or "transformational"?
- [ ] Any phrases that sound like they came from a template?
- [ ] Is every sentence substantive with real meaning?

### UNINTENDED IMPLICATIONS
- [ ] Does "managed" imply hands-on when it was delegated?
- [ ] Does "led" overstate individual contribution?
- [ ] Are there claims that could be questioned in interview?
- [ ] Is scope properly bounded (function vs enterprise)?
- [ ] Any accidental overselling?

### NATURAL LANGUAGE CHECK
- [ ] Read summary aloud - does it sound like a human wrote it?
- [ ] Read each bullet - natural sentence structure?
- [ ] Variety in sentence length and structure?
- [ ] Would you say these things in conversation?

### EXECUTIVE LENS
- [ ] Does this position Jeremy at the right level?
- [ ] Are achievements appropriate for Director/VP level?
- [ ] Too much tactical detail vs strategic outcomes?
- [ ] Leadership positioning clear?

### SOUNDBOARD LENS
- [ ] What questions might this resume raise?
- [ ] What's the strongest part? The weakest?
- [ ] What would make it better?
- [ ] Any red flags for hiring manager?

## COMMON ISSUES TO CATCH

**Issue 1: Combining Separate Projects**
❌ "Led platform transformation and pricing optimization delivering $50M"
✓ Two separate bullets for two separate achievements

**Issue 2: Scope Creep**
❌ "Set enterprise AI strategy" (when it was just your function)
✓ "Developed AI strategy for commercial platforms"

**Issue 3: AI Voice**
❌ "Spearheaded transformational initiative leveraging cutting-edge technologies"
✓ "Led platform migration achieving 50% cost reduction"

**Issue 4: Undefendable Claims**
❌ "Architected cloud-native microservices" (if you didn't architect it)
✓ "Led team that architected cloud-native microservices"

## GATE CONDITION
**MUST pass logical consistency check to proceed**

If issues found:
1. Note specific problems
2. Fix them
3. Re-read
4. Proceed to Agent 6 when clean

## OUTPUT FORMAT

✅ AGENT 5 PASSED - Logically consistent, no AI garbage
OR
⚠️ AGENT 5 CONCERNS - [list issues] - Recommend fixes

## VALIDATION CHECKPOINT
✓ Read aloud check complete
✓ Logical consistency verified
✓ AI garbage removed
✓ Implications checked
✓ Ready for Agent 6
