## HOW TO USE THIS DOCUMENT

### When Creating Tailored Resumes:
1. Agent 4 will check EVERY number, date, and title against this document
2. If something doesn't match → Agent 4 fails → resume gets fixed
3. This ensures you never accidentally fabricate or misremember facts

### When to Update:
- New job or promotion → Add new role section
- Completed major achievement → Add to Key Metrics
- Career value increases → Update Cumulative Value section
- You realize you're phrasing something inconsistently → Add to Terminology Rules

### Accuracy Standards:
- Conservative estimates only (if $27M-$35M range, document $27M)
- Round to nearest significant figure ($3.2M not $3,247,891)
- Always note timeframe (annual vs cumulative vs one-time)
- Only include what you can defend in an interview
